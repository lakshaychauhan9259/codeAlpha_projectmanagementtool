TaskFlow – Project Management Application

TaskFlow is a modern, single-page project management and team collaboration application designed to help teams organize projects, manage tasks, track progress, and improve productivity through an intuitive Kanban-based workflow. The application features a sleek dark-themed interface and provides essential project management tools in a single, responsive web application.

Getting Started
Download the project files.
Open project.html in any modern web browser.
Log in using the sample accounts or create a new account.
Start creating projects and managing tasks.

No additional installation, build tools, or dependencies are required because the entire application is contained within a single HTML file.
